#next ⇒ Lesson?

Returns the next Lesson, or `nil`.

The next lesson is defined as the one with the next number (i.e. current number plus 1), so if there's a “gap” (e.g. lesson numbers 1, 2, 4), `nil` will be returned instead.

Examples:

Getting lesson 2

Lesson.first.next #=> #<Lesson number=2>

  • (Lesson, nil)
